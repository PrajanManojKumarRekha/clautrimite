# Security Policy

## Reporting A Vulnerability

If you believe you have found a security issue in Clautrimite, please do not open a
public issue with exploit details.

Instead:

1. describe the vulnerability clearly
2. include affected files or commands
3. include reproduction steps if possible

Open a private security report through GitHub if available for this repository, or
contact the maintainer directly through the repository owner profile.

## Scope

Clautrimite is a routing and orchestration layer for Claude Code.

Security-relevant issues may include:

- unsafe installer behavior
- misleading command routing that could cause destructive workflow mistakes
- incorrect setup guidance that weakens secret handling or execution safety

Issues in upstream frameworks should also be reported to their respective maintainers when
appropriate.
